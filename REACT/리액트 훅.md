## useState
가장 기본적인 훅으로, 함수 컴포넌트에서 상태를 관리 할 수 있다.

```js
import { useState } from "react";

const Counter = () => {
    const [value, setValue] = useState(0);

    return(
        <div>
            <p>숫자는 <p>{value}</p>이다</p>
            <button onClick={()=>setValue(value+1)}>+1</button>
            <button onClick={()=>setValue(value-1)}>-1</button>
        </div>
    );
};

export default Counter;
```
버튼에 따라 숫자가 늘었다 줄었다 하는 코드
